# JumpSaaS

Production-ready Next.js 15 SaaS template — everything you need to launch your SaaS product, nothing you don't.

## What's included

- **Auth** — Sign up, sign in, 2FA, email verification via Better Auth
- **Billing** — Stripe subscriptions, one-time payments, customer portal
- **i18n** — Multi-language support with next-intl (English + German out of the box)
- **Email** — Transactional emails with React Email
- **File uploads** — S3-compatible storage
- **Database** — PostgreSQL + Drizzle ORM with a clean schema convention
- **Admin dashboard** — User and subscription management
- **Dark mode** — System-aware theming

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS v4 · Better Auth · Stripe · Drizzle ORM · PostgreSQL · next-intl
